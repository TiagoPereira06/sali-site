Sistema de Apoio a Lares de Idosos

O objetivo é ajudar não só à gestão remota do lar por parte do seu administrador, como também monitorar as interações dos auxiliares e enfermeiros com os pacientes.

Relatório Final Proposta Inicial
Funcionalidades

Pensado para o bem-estar dos pacientes

Para Administradores e Funcionários :

Confirmação de identidade

Se necessário, os funcionários podem confirmar a identidade (pulseira NFC) do paciente.

Alertas em tempo real

Em caso de incidente, o administrador é notificado do sucedido, em tempo real.

Diagnóstico de doenças

Usando as capacidades do módulo de IA é possivel prever possíveis doenças.

Web API

Suporta operações lógicas sobre as várias entidades da solução.

  • Baseado na Clean Architecture
  • Padrão CQRS
